Detailed Description

This class represents a value from the grammar (https://www.json.org/). Different from the grammar, this class also acts as a member and an element. It acts as a member by storing the associated string (name) to the value. An element is simply a value surrounded by optional whitespace. This class keeps track of that whitespace acting as an element as well.
